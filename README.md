# post-for-me-skills

Agentic skills for the Post For Me API:
https://github.com/DayMoonDevelopment/post-for-me

This repository is a small collection of reusable `SKILL.md` documents that teach coding agents how to work with Post For Me (SDK usage, API flows, payload shapes, etc.).

## Install

Install this skill pack with the `skills` CLI:

```bash
npx skills add DayMoonDevelopment/post-for-me-skills
```

## Repo layout

- `skills/<skill-name>/SKILL.md`: A single skill document (YAML frontmatter + markdown)
- `AGENTS.md`: Authoring conventions for skills in this repo

## Available skills

- `post-for-me-ts-sdk`: TypeScript SDK usage guide (`skills/post-for-me-ts-sdk/SKILL.md`)
